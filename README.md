# Ecommerce Demo

This repository is a demo project for an event-driven e-commerce solution built with .NET.

## IMPORTANT NOTE!

All secrets shown in this repository, including passwords and connection strings, are present only for demo purposes so the project is easy to run locally.

In a real project, these values should be handled properly through secure secret management such as secret vaults and environment-specific configuration.

Detailed documentation is kept in the `docs` folder. See:

- `docs/docker-compose.md`
- `docs/gateway.md`
- `docs/scripts.md`
- `docs/technologies.md`
